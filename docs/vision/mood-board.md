# Mood Board & Visual References

Design references for Cineborough UX. Assets live outside the repo (screenshots, vision board images).

## Phase 1 — Reventure-Style Data Engine

- Choropleth map with sidebar metric layer picker
- Zip-level signals: Forecast & Valuation card + Demographics card
- Opportunity Index as default heatmap layer
- Dark UI with high-contrast metric typography

## Phase 2 — Cinematic UX (Deferred, ADR-008)

- Google Earth-style descent into neighborhood
- Scroll-driven camera paths (GSAP ScrollTrigger)
- Locale quote cards over blurred city photography
- Glowing route lines for transit and greenways
- Transition from 3D tiles to static 4K neighborhood photography
